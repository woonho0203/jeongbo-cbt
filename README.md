# 정보처리기사 필기 CBT 연습 시스템

본인 보유 PDF 자료(2024-2025년 6회차 기출 + 5개 유형별 자료)에서 자동 추출한 문제로 구성된 CBT(Computer-Based Test) 시스템.

## 주요 기능

- **회차별 기출문제 풀이**: 2024년 1·2·3회, 2025년 1·2·3회 (회차별 약 100문제, 150분 타이머)
- **랜덤 모의고사**: 전체 기출에서 무작위 100문제 추출
- **유형별 연습 모드**: 필수계산 41 / 필수코드 56 / 잘못된문장 197 / 키워드 259 / 종류·순서 114 (해설 포함, 학습용)
- **오답 노트**: 직전에 틀린 문제만 모아서 다시 풀기
- **북마크/별표**: 시험 중 ⭐ 버튼으로 다시 보고 싶은 문제 표시
- **OMR 답안 현황**: 우측 패널에서 응답·미응답·북마크 상태 확인 + 빠른 이동
- **과목별 점수 분석**: 5과목 정답률 막대그래프 + 합격선(60점) 표시
- **시험 기록**: 모든 응시 결과를 JSON 파일에 저장, 상세 결과 재조회 가능
- **학습 통계 대시보드**: 평균/최고/최근 점수, 점수 추세 SVG 차트, 과목별 평균
- **키보드 단축키**: 1~4 (보기 선택), ←/→ (문제 이동)

## 실행 방법

### 사전 요구
- Node.js 18+ (`node -v`)
- (PDF 재추출 시) Python 3 + `pdfplumber` (이미 추출된 JSON이 포함되어 있어 일반 사용 시 불필요)

### 한 번만 설치 (의존성 다운로드)

```bash
cd cbt-app
npm install
```

### 서버 시작

```bash
npm start
# 또는
./start.sh
```

브라우저에서 http://localhost:3000 접속.

### 데이터 재추출 (선택)

PDF 원본을 수정했거나 다른 PDF를 추가했을 때:

```bash
pip install pdfplumber
python3 scripts/extract_answers.py
python3 scripts/extract_questions.py
python3 scripts/extract_categories.py
```

## 사용 팁

- **시험 중 이탈 주의**: 새로고침하면 진행 상황이 초기화됩니다.
- **자동 채점은 회차별/랜덤만 적용**: 유형별 자료는 정답이 명시되지 않은 경우가 많아 학습용으로 동작합니다(해설 표시).
- **합격 기준**: 평균 60점 이상 + 모든 과목 40점 이상 (정보처리기사 필기 기준). 결과 화면에서 자동 표시.

## 데이터 저장

응시 기록·오답·북마크는 모두 `data/cbt_store.json` 파일 한 개에 저장됩니다.
이 파일만 백업/복원하면 모든 학습 이력이 보존됩니다.

## 의존성

`express` 한 개뿐. 네이티브 빌드가 없어 macOS·Linux·Windows 어디서든 `npm install` 한 번으로 끝납니다.
